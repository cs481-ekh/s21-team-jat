# ALD Process Monitoring Web Application
Before starting make sure you have the latest stable version of Node.js ([14.15.4](https://nodejs.org/en/download/)) installed.
## Building
To build, run:
```bash
npm install
```
or:
```bash
npm run build
```

## Running
Production:
```bash
npm start
```
Development:
```bash
npm run dev
```
